This application has been built to log, modify and delete users & assets. Additionally, assets can be assigned to users.

Application information:
This app utilises Flask in python, css (Jinja2), HTML and SQLite. The project artefacts can be found in this file repository/zip files in order to recreate locally. 
The production URL for the app is: https://software-engineering-agile-app.onrender.com/

Profiles:
Functionality is defined by roles. Administrators have full create, read, update and delete permissions, whereas users cannot delete records.
